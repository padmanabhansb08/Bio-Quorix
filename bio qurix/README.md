# BioQuorix

Static frontend (HTML/CSS/JS) served by a tiny Node server.

## Project structure

- `public/`: web app (open in browser)
  - `public/index.html`
  - `public/css/style.css`
  - `public/js/app.js`
  - `public/js/data.js`
  - `public/assets/logo.jpg`
- `server.js`: local dev server (serves `public/`)

## Run locally

```bash
node server.js
```

Then open `http://localhost:8000/`.
